# Electronics Customer Analytics
Customer analytics dashboard built with **Python (Pandas)** and **Tableau Public**.

### Dashboard

![Dashboard](images/dashboard_prerview.png)

The screenshot demonstrates interactive filtering by:
- Year
- Category

## Project
In this project I:
- cleaned raw data
- merged multiple datasets
- calculated customer metrics
- prepared data for Tableau
- built an interactive dashboard

## Dashboard
The dashboard includes
- Revenue
- Customers
- Orders
- Average Check
- Revenue by Category
- Revenue by Month
- Revenue by City
- Revenue by Payment Method
- Top 10 Customers by LTV

Interactive filters:
- Year
- Category
- Payment Method
- City

## Tech Stack

- Python
- Pandas
- Tableau Public
